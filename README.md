# 🖐 Hand Gesture Recognition System

> Show your hand to the camera — it tells you exactly what gesture you're making, in real time.

A real-time hand gesture recognition system that uses your webcam to detect and classify hand gestures using MediaPipe's Hand Landmarker API and OpenCV. Detects 21 hand landmarks per frame and maps finger positions to gesture names and actions.

Built with Python — no deep learning training required.

---

## ✨ Features

- 🤚 **7 Gesture Types** — Fist, Open Palm, Thumbs Up, Peace, Pointing, Rock, Four Fingers
- ⚡ **Real-time Detection** — runs live through webcam feed
- 🖼️ **Visual Overlay** — draws hand skeleton with landmarks on screen
- 🏷️ **Gesture + Action Display** — shows detected gesture and mapped action on screen
- 🤜 **Left & Right Hand Support** — auto-detects hand orientation
- 🪶 **Lightweight** — no TensorFlow training, uses pretrained MediaPipe model

## 🎯 Gesture Map
